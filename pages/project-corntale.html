CornTale Shop Sim!

Undertale's new fan location! 2D Unity ambition project


Bit of history:

This was made as a project for my software development study to show off what development skills I learned during the first half of my first study year here!

In the first year I was aiming to the highest grade on every big project, but unfortunately I had a fall-out for a few weeks when I got a new Unity 2D project, and I didn't succeed with making it better than just sufficient.
However my chance arrived again when we got an opportunity to make whatever we want, for a new Ambition project, as long as it fits within the study.


Dev process and End Goal:

The plan was to create a simulation of a shop, with functions directly taken from the inspiration source:
- Player can buy 4 different items, the first 2 can only be bought once.
- Player has an inventory that fills up (only visually)
- Player has a certain amount of starting money (G) that gets affected after purchase.
- Player can talk to the shopkeeper on 4 different topics.
- Player can try to sell things to the shopkeeper, but they will refuse.
- There is an exit button that restarts the game, instead of exiting it.






Conclutions and Learned skills:

The first reason to make this project was to get more familiar with Unity, and I can proudly say that it worked! I am quite comfortable using Unity for things like this. This project also improved my C# skills for sure! It was also a head start to working in new medium -> Pixel Art.
