CornTale Shop Sim!

Undertale's new fan location! 2D Unity ambition project


Bit of history:

CornTale was made as a project for my software development study to show off my developer skills I got during the first half a year here.

I fumbled on one of the previous Unity 2D projects, and I needed to redeem myself.
Since this was an abition project, it meant I could make whatever I want as long as it fits withing the study. And "2D game" + "whatever I want" = Undertale Fangame.

You can download or play the game in web here on Itch.io



Styling choices and End Goal:

The plan was to create a simulation of a shop, with functions directly taken from the inspiration source:
- Player can buy 4 different items, the first 2 can only be bought once.
- Player has an inventory that fills up (only visually)
- Player has a certain amount of starting money (G) that gets affected after purchase.
- Player can talk to the shopkeeper on 4 different topics.
- Player can try to sell things to the shopkeeper, but they will refuse.
- There is an exit button that restarts the game, instead of exiting it.


Conclusions and Learned skills:

Every single point from above was achieved!! The first reason to make this project was to get more familiar with Unity, and I can proudly say that it worked!

I've also improved my research skills, since I would only follow guides I'd find myself. But I did still ask for a lot of feedback from my friends and teachers.
I am quite comfortable using Unity for things like this now. This project has also improved my C# skills and was a head start for me to get into the new medium -> Pixel Art.

For this version of the game (v1.0) I must say I had a terrible time with Unity Animator, so sometimes the game would break if spammed too much (that was fixed in the later update)
